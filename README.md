<h1>Password Manager</h1>
A password manager app for websites.
<br />

<h2>Languages and Utilities Used</h2>

- <b>Python</b> 
- <b>Packages - Tkinter, Pyperclip, Json</b>
<br />

<h2>Program walk-through:</h2>
<br/>
<p align="center">
Type in the website name and email id/username.
